Resolume Arena is the standard-bearer for live visual performance — the software running behind LED walls at club nights, festival stages, theatre productions and brand activations worldwide. Arena adds the full output mapping engine on top of Avenue's clip playback, turning any screen array into a single addressable canvas.

### Layer stack model

```
Composition
└── Layer N (blend mode · opacity · effects chain)
    └── Clip [video / image / generator / live input]
         └── Effect chain [per-clip FX parameters]
```

### Connectivity

| Protocol | Use |
|---|---|
| DMX (ArtNet · sACN) | Lighting console sync |
| MIDI | Hardware controller / launchpad mapping |
| OSC | Generative software (TouchDesigner, Max/MSP) |
| Spout / Syphon | Real-time texture sharing with 3D engines |
| NDI | Network video over LAN |
| FFGL | Custom GPU shader plugins |

### Arena-exclusive: Advanced Output

- Edge blending for multi-projector soft-edges
- Slice transformation — map output to any polygon shape
- Fixture LED mapper — imports rig layout from MA / Capture
- Soft-edge overlap with gamma-corrected blend zones
- Per-screen colour calibration

### Built-in generators

Solid · Gradient · Noise · Lissajous · Plasma · Tunnel · Strobe · LFO visualiser · Audio reactor
